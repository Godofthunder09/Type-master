# ⌨️ TypeMaster — Professional Typing Platform

A free, no-login typing practice platform built with pure HTML, CSS, 
and Vanilla JavaScript. No frameworks. No dependencies. Just fast, 
feature-rich typing practice.

🔗 Live Demo: https://typemaster.42web.io/

---

## ✨ Features

- 🕐 **Typing Test** — Time mode (15s/30s/60s/120s), Words mode, 
  and Pro Mix (words + numbers + symbols)
- 📊 **Live Stats** — Real-time WPM, accuracy, error count, 
  and Canvas-based WPM graph
- 🔥 **Keyboard Heatmap** — Visual error frequency map 
  per key to identify weak spots
- 📖 **8 Structured Lessons** — From Home Row basics to 
  Professional Email Phrases
- 🎮 **3 Typing Games** — Word Rain, Speed Rush, Precision Strike
- 💀 **Special Modes** — Blind Mode, Sudden Death, 
  Reaction Mode, Rhythm Mode
- 🎵 **Rhythm Mode** — BPM-synced beats using Web Audio API 
  (Drum, Piano, Electronic)
- 🌍 **7 Languages** — English, Hindi, Marathi, French, 
  Spanish, Code Snippets, Numbers
- 💾 **100% Private** — All data stored in localStorage, 
  no account needed

---

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Structure | HTML5 |
| Styling | CSS3 (custom, no framework) |
| Logic | Vanilla JavaScript |
| Audio | Web Audio API |
| Graphics | HTML5 Canvas API |
| Storage | localStorage |

## 🚀 How to Run

No installation needed.

1. Clone the repository
   git clone https://github.com/Godofthunder09/type-master.git

2. Open index.html in any browser

That's it. No npm install. No build step.
